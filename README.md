# stock-price-movement
stock price movement display

## Prerequisites
* Maven 3.6.1
* JDK >= 8
## Technologies
* Spring ( Spring-boot, Webflux )
* SSE ( Server Sent Event )
* Angular

## How to run it
* mvn clean spring-boot:run
* open browser http://localhost:5000/
